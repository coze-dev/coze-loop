# @cozeloop/adapter-interfaces

CozeLoop adapter interfaces

## Overview

This package is part of the Coze Loop monorepo and provides adapter interfaces definition.

## Getting Started

### Installation

Add this package to your `package.json`:

```json
{
  "dependencies": {
    "@cozeloop/adapter-interfaces": "workspace:*"
  }
}
```

Then run:

```bash
rush update
```

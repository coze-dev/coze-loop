### 🟢 轻微问题 #1 [文案差异]

**位置**：第 3 节 交互流程图

**问题**：交互流程图中第一步写的是"用户进入 Prompt 列表页面"，但 PRD 原文为"用户进入 Prompt 开发页面"，文案不一致。

**修复**：将"Prompt 列表页面"修改为"Prompt 开发页面"，与 PRD 保持一致。

---

### 🟡 中等问题 #1 [逻辑缺失]

**位置**：第 7 节 业务逻辑规则（新增 BL-004）

**问题**：Spec 未说明当默认 Prompt Key `prompt_key_0` 已被占用时的处理方式。用户不修改默认值直接提交可能遇到重复 key 错误。

**修复**：新增 BL-004 业务逻辑规则，以 `[待澄清]` 标记标注此场景，需产品确认处理方式。
